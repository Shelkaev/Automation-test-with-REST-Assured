Проект с API тестами на основе сервиса https://reqres.in
Адрес тестируемого ресурса подгружается из файла конфигурации config.properties
Прект поделен на 4 тестовые группы для каждого эндпоинта:
 - LoginRestTest
 - RegistrationRestTest
 - ResourceRestTest
 - UserRestTest
 - RunAllTests - запускает все тесты
